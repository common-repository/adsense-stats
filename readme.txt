Widget for WordPress to display detailed statistics about your Google Adsense earnings.

== Description ==

Adsense Stats is a widget for WordPress to display detailed statistics about your Google Adsense earnings without requiring login into your Google account. 

== Installation ==

1. Upload adsense-stats directory to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure and enjoy :)

Once the plugin is activated it creates a "AdSense Stats" panel under "Options". It will allow you to manage the Google Adsense login.
